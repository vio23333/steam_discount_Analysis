📊 Steam 特價遊戲資料分析專題
專案簡介

本專題以 Steam 特價遊戲資料 為研究對象，透過 Selenium 爬蟲 收集遊戲特價資訊，並使用 Python 與 Excel 進行資料清洗、特徵工程與視覺化分析，探討：

"折扣等級與遊戲原價的關係"
"折扣等級與上市時間（年）的關係"
"折扣等級與使用者評價等級的關係"

本專案主要目標為展示 資料蒐集 → 清洗 → 分析 → 視覺化 的完整流程。

使用技術

Python
Selenium（網頁爬蟲）
Pandas（資料清洗與轉換）
Excel
樞紐分析表（Pivot Table）
圖表與互動式篩選
資料處理技術
折扣／價格分級（Discretization）
發售日期轉換（年／季度）
類別型資料數值化（評價等級）

資料來源

Steam 官方商店特價頁面
（僅用於學術與個人作品展示，未作商業用途）
